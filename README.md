# 121Gramify

A modern English Learning Android application built with **Kotlin**, **Jetpack Compose**, **MVVM**, and **Clean Architecture**.

121Gramify helps users improve their English language skills through grammar lessons, vocabulary building, dictionary support, quizzes, and progress tracking in a clean and user-friendly interface.

---

## Features

### 📚 Grammar Learning

* Learn English grammar rules step-by-step
* Beginner to advanced grammar concepts
* Easy-to-understand explanations

### 📖 English Dictionary

* Search meanings of English words
* Pronunciation support
* Parts of speech and usage examples

### 🌟 Word of the Day

* Learn a new word every day
* Improve vocabulary consistently

### 📝 Practice Quizzes

* Test grammar and vocabulary knowledge
* Interactive learning experience

### 📈 Progress Tracking

* Monitor learning progress
* Daily goals and achievements

### 🎨 Modern UI

* Built with Material Design 3
* Responsive and intuitive user experience
* Jetpack Compose-based UI

## Architecture

The project follows **Clean Architecture** and **MVVM** principles.

```text
presentation/
├── feature
├── navigation
├── components

domain/
├── model
├── repository
├── usecase

data/
├── repository
├── model
├── mapper
├── datasource

core/
├── utils
├── constants
├── common
```

## Getting Started

### Prerequisites

* Android Studio Meerkat or newer
* JDK 17+
* Android SDK 24+

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/121Gramify.git
```

2. Open the project in Android Studio

3. Sync Gradle

4. Run the application
